    Upload Instructions

    Supported File Types

    • Documents: PDF, DOC, DOCX, TXT, RTF, ODT, EPUB
    • Presentations: PPT, PPTX, KEY, ODP
    • Spreadsheets: XLS, XLSX, CSV, ODS
    • Images: JPG, JPEG, PNG, GIF, SVG, BMP, TIFF
    • Videos: MP4, AVI, MOV, WMV, FLV, MKV, WEBM
    • Audio: MP3, WAV, AAC, FLAC, M4A
    • Code Files: JS, HTML, CSS, PY, JAVA, C, CPP, PHP, SQL
    • Archives: ZIP, RAR, 7Z, TAR, GZ
    • And many more... Almost any file type is accepted!

    Important Notes

    • There is NO file type restriction - upload any educational material
    • Files will appear in the "Resources" page immediately
    • Files are stored on the server in your personal folder
    • Maximum file size: 2GB
    • Large files (>10MB) are uploaded in chunks for reliability
    • Make sure files don't contain sensitive personal information
    • All uploaded files are visible to everyone by default (unless marked as private)

    Best Practices

    • Use clear, descriptive titles for easy searching
    • Add detailed descriptions to help classmates understand the content
    • Select appropriate category and resource type for better organization
    • For multiple related files, consider zipping them together
    • Check file names for any typos before uploading
    • Public resources become available to everyone immediately
    • Videos and images can be viewed directly in the Resources page
